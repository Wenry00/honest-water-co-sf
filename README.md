# Honest Water Co SF Website

Deploy this project on Vercel.

## Fast deploy steps
1. Create a free GitHub account if you do not have one.
2. Create a new GitHub repository named `honest-water-co-sf`.
3. Upload everything from this folder into that repository.
4. Go to Vercel.com and sign in.
5. Click `Add New Project`.
6. Import the GitHub repository.
7. Framework preset should detect `Vite`.
8. Build command: `npm run build`.
9. Output directory: `dist`.
10. Click Deploy.

## Images
All site images are already wired inside `public/images`.

## GoHighLevel connection
The current quote form is visual only. Replace the form area with your GoHighLevel embed code when your CRM is ready.
