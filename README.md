# Golden Crest Water Website

Vite/React landing page for Golden Crest Water.

## Deploy on Vercel
1. Upload this folder to a GitHub repo.
2. Import repo in Vercel.
3. Framework: Vite.
4. Build command: `npm run build`.
5. Output directory: `dist`.
